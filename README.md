## Development

We use `node` version `14.9.0`

```
nvm install 14.9.0
```

```
nvm use 14.9.0
```

The first time, you will need to run

```
npm install
```

Watch typescript with

```
npm run watch
```

Then just start the server with

```
npm run dev
```

It uses nodemon for livereloading
